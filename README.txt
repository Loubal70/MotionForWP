This plugin adds a new features to the Gutenberg editor. Allows you to add a motion in your page.

== Description ==

This plugin is a powerful extension for the renowned Gutenberg editor in WordPress. It empowers you to seamlessly
incorporate captivating animations into your web pages, enriching the visual experience for your visitors. With an
intuitive interface, you can easily customize animation speed, direction, and triggers for various content types, from
images and videos to text and media blocks.

== Installation ==

1. Search for the plugin in WordPress under "Plugins -> Add New".
2. Click the “Install Now” button, followed by "Activate".

That's it!
